# Stock Philosophy ～ ふたつの世界線 ～

株式会社Stockが大切にしている10の考え方を、パラレルワールドの探索を通じて学ぶオンボーディング教材です。

## 🎮 遊び方

[▶️ ゲームをプレイする](https://shunichi-takeda.github.io/stock-philosophy-game/)

1. 「世界線B（考え方のない世界）」と「世界線A（考え方のある世界）」を行き来する
2. ふたつの世界の違いを見つける
3. 自分自身の経験を振り返る
4. Stockの実話エピソードを知る

## 📚 学べるフィロソフィー

| # | 考え方 |
|---|--------|
| 1 | 「心身の健康」よりも大切なタスクは、絶対に存在しない |
| 2 | 「本人に操作性のない事柄」での差別は、絶対に禁止 |
| 3 | 40度の熱があっても、0.5秒で分かるのか？ |
| 4 | 圧倒的なスピードは、相手の心を打つ |
| 5 | 「Proactive（能動的）」なアクション |
| 6 | サンクコストを切り捨てろ |
| 7 | 全員がフェアであるために「論理的」であることを重視 |
| 8 | 「すみません」から入れるか |
| 9 | グレーが最悪 |
| 10 | 「自分たちが存在する世界線」と「存在しなかった世界線」の差分を広げる |
